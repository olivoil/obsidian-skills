# skills

Personal cross-agent skills for recurring work.

These are written for my tools and workflows. Some may be useful as-is; others are better treated as examples to adapt.

## Skill groups

- [`skills/obsidian`](./skills/obsidian/README.md) — workflows for maintaining my Obsidian vault.
- [`skills/engineering`](./skills/engineering/README.md) — engineering workflows such as PR review and security incident response.

## Install

```bash
npx skills add olivoil/skills
```

For a local checkout:

```bash
npx skills add ~/Code/github.com/olivoil/skills
```

See the group READMEs for workflow-specific notes.

## Update

```bash
npx skills update
```
